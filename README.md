## 🚀 طريقة الرفع على Railway (خطوة بخطوة)

### الخطوة 1 — إنشاء حساب GitHub
1. افتح **github.com** وسجّل حساباً مجانياً
2. اضغط **New Repository** وسمّه `sahm-platform`
3. اضغط **Create Repository**

### الخطوة 2 — رفع الملفات
1. في صفحة الـ Repository اضغط **uploading an existing file**
2. اسحب جميع ملفات هذا المجلد وأسقطها في الصفحة
3. اضغط **Commit changes**

### الخطوة 3 — الربط مع Railway
1. افتح **railway.app** وسجّل دخول بحساب Google
2. اضغط **New Project** ثم **Deploy from GitHub repo**
3. اختر مستودع `sahm-platform`
4. Railway سيبني المشروع تلقائياً ✅

### الخطوة 4 — ربط الدومين
1. في Railway افتح مشروعك
2. اضغط **Settings** ثم **Domains**
3. أضف دومينك الخاص
